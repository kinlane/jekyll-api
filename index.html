Jekyll API

This is playground to figure out various ways to use Github Jekyll to emulate API behavior. There are many different approaches, but I want to isolate ways that are 100% Github ran, without outside dependencies.

My first example uses Jekyll for managing the content of an articles API.

  • Articles

Each article is a single Jekyll post, with the title as YAML front end matter, and content is the body of the post.

Next I will play with using categories and tags to better filter searches.
